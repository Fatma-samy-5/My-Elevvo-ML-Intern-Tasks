# 🤖 Elevvo Machine Learning Internship

This repository contains the projects and technical tasks I completed during my **Machine Learning Internship at Elevvo**.
The internship focused on building end-to-end ML pipelines, from data preprocessing and exploratory data analysis (EDA) to advanced predictive modeling and recommendation systems.

🚀 **Overview**

During this internship, I worked on several hands-on projects to strengthen my understanding of:
- **Supervised Learning**: Building regression and classification models.
- **Unsupervised Learning**: Implementing clustering algorithms for customer insights.
- **Recommendation Engines**: Creating personalized user experiences using collaborative filtering.
- **Time Series & Forecasting**: Predicting future trends and sales using advanced ensemble methods.
- **Data Engineering**: Cleaning, scaling, and feature engineering for real-world datasets.

📌 **Skills Gained**

- Developing predictive models using **Scikit-learn, XGBoost, and LightGBM**.
- Implementing **Customer Segmentation** using K-Means clustering.
- Building **Recommendation Systems** based on user-item correlations.
- Performing **Time Series Forecasting** for sales and business metrics.
- Data visualization and storytelling using **Matplotlib and Seaborn**.
- Version control and collaborative development with **Git & GitHub**.

🗂 **Internship Tasks**

Here are the main tasks I completed during the internship:

* **Task 1: Student Score Prediction**
    * Linear Regression model to predict student scores based on study hours.
* **Task 2: Customer Segmentation**
    * Unsupervised learning using K-Means to group customers based on purchasing behavior.
* **Task 5: Movie Recommendation System**
    * Collaborative filtering system using the MovieLens 100K dataset to suggest movies.
* **Task 7: Sales Forecasting**
    * Time-series regression pipeline using XGBoost and Random Forest to predict future sales.
* **Task 8: Traffic Sign Classification**
    * Multi-class classification project using MobileNet and Transfer Learning to recognize traffic signs from the GTSRB dataset.
🛠 **Technologies Used**

- **Python** (Pandas, NumPy, SciPy)
- **Machine Learning**: Scikit-learn, XGBoost, LightGBM
- **Visualization**: Matplotlib, Seaborn
- **Tools**: Jupyter Notebooks, VS Code
- **Version Control**: Git & GitHub
